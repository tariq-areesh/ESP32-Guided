Getting Started

Follow these steps to prepare your environment and run the Blink example. The links below go to official resources.

Install the Arduino IDE

  1. Download from arduino.cc/en/software.
  2. Install and open the IDE once.

Add ESP32 boards

  1. Open File → Preferences and paste: https://espressif.github.io/arduino-esp32/package_esp32_index.json
  2. Open Tools → Board → Boards Manager, search esp32, install "esp32 by Espressif Systems".

Docs: github.com/espressif/arduino-esp32

USB drivers

  • CP210x: silabs.com
  • CH340/CH341: wch-ic.com

First program: Blink

  1. Open File → Examples → 01.Basics → Blink.
  2. Set LED pin to 2 for many boards.
  3. Upload. If needed, hold BOOT while connecting.
  4. Open Serial Monitor for logs.
